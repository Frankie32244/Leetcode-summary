  动态规划比较难，数据结构的题目较简单，前缀和，滑动窗口，哈希表等类型的题目经常一块使用。位运算总是搞得我头皮发麻。
  也算是马马虎虎地过了算法这一关，面试的话突击突击应该问题不大。经过一年多的熏陶，每每没有思路的时候，痛苦地抓头发，但是回顾下来，好好debug,或者认真看别人的题解。也能从中学到很多，还是那句话。坚持下来，能收获很多东西！加油！打工人！
